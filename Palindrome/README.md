# Palindrome in Swift!

this is a demo application where you can:
- insert a word,
- check if the word is a palindrome
- add the word to a database.
- view all stored words 
- remove the word from a database.

Although simple we use:
- MVVM
- Box Bind
- Unit tests and UI tests

CocoaPods:
- Realm
- Swinject
- Quick / Nimble
- IQKeyboardManager
